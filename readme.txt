PayTR mağazanıza ait taksit seçeneklerinizi ürün sayfasında göstermenizi sağlayan eklenti.

== Description ==
PayTR mağazanıza ait taksit seçeneklerini ürün detay sayfasına yeni bir **tab** ekleyerek göstermenizi sağlar. Seçenekleri düzenleyebilir, açıklama ekleyebilirsiniz.

**Bilgi:** PayTR WooCommerce Taksit Tablosu yalnızca iFrame API Entegrasyonu ile birlikte çalışmaktadır.

[PayTR Sanal Pos ve Ödeme Hizmetleri](https://www.paytr.com/)

= Gereksinimler =
* WooCommerce eklentisi ile birlikte PayTR’da bir mağazanız olması gerekmektedir.

== Installation ==
1. İndirdiğiniz dosyadan çıkardığınız klasörü "/wp-content/plugins/" klasörüne yükleyin.
1. Eklentiler menüsünden **PayTR Taksit Tablosu - WooCommerce** eklentisini etkinleştirin.
1. **PayTR Taksit** menüsünden gerekli düzenlemeleri yaparak kullanmaya başlayabilirsiniz.
